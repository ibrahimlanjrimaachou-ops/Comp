Tarifs officiels

Grille tarifaire BruBasket 2026
Type Description Prix
Ticket journalier Accès à une journée de compétition 10€
Pass événement Accès aux six jours du tournoi 45€

Les billets sont disponibles en ligne et sur place dans la limite des places restantes.

Conseils d'achat

Pour la finale

Réservez votre place en avance, la finale attire le plus grand nombre de visiteurs.

Pour tout le tournoi

Le pass événement est la formule la plus avantageuse si vous souhaitez revenir plusieurs jours.

Pour les clubs

Les groupes et partenaires peuvent contacter l'organisation pour un accompagnement dédié.
